- Erro na comunicação com a API: deve ser apresentada a mensagem de erro correspondente.

- Classes:
    - View/Interface do usuário
    - Controller - trata os dados da view e envia para o model
    - Model - trata os dados de acordo com a regra de negócio
    - Conversor monetário - classe que converte o valor de uma moeda para outra
    - API - classe que faz a comunicação com a API de conversão monetária
    - Classe de validação para fazer a conversão monetária
    - Classe para tratamento de erros do Conversor
    - Classe para tratamento de erros da API
    - Classe para tratamento de erros da validação
    - Cada classe tem uma interface
